# Object Types / Data Types

- Number : 1234, 3.14, 3+5j, 0b11, Decimal(), Fraction()

- String : 'spam', "Bob's" etc...

- List : [1, [2, 'three'], 4.5], list(range(10))

- Tuple : (1, 'spam', 4, 'U'), tuple([1, 2, 3])

- Set : {1, 2, 3, 4}, set('spam')

- Dictionary : {'food': 'spam', 'taste': 'yum'}, dict(hours=10)

- File : open('eggs.txt')

- Boolean : True, False

- None : None

- Functions, modules, classes

- Advance : Decorators, Generators, Iterators, MetaProgramming7
